Pharmacy Expense Tracker - Build Instructions
========================================

1. Extract this ZIP file to a folder on your Windows computer
2. Right-click on 'build_windows.bat' and select 'Run as administrator'
3. Wait for the build process to complete
4. The final executable package will be created in the 'dist' folder
5. The final ZIP file can be distributed to other Windows computers

Requirements:
- Windows 7 or later
- Python 3.7 or later
- Internet connection (for downloading dependencies)
